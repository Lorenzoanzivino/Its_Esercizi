Introduzione a FLASK:
    Per sviluppare la logica dell'applicazione web sfruttando ad esempio Python, Le web app basate su modello client-server.
    Model View Controller (MVC):
        Modello su cui si basa una web app
            - view = user interface
            - controller = application logica
            - model = DB strutture e query
    
    Flask è un micro framewark scritto in python, reso flessibile e leggero, lasciando al programmatore la possibilità di scegliere con quali librerire lavorare. 

    Come installare flask:
        - Versione python >= 3.9
        - Scrivere su terminale "pip install Flask"
        - Nel nostro caso con i nostri pc "sudo apt install python3-flask"
        - Nel mio pc in particolare mi sposto nella cartella vscode_projects e poi "pip install Flask"

        - Dopo in un file.py scrivere :

            from flask import Flask
            
            app = Flask(__name__)

            #decoratore
            @app.route('/') # avere solo il percorso e basta restituisce un get
            def home() -> str:
            return "<h3>Hello, world!</h3>"

            # sempre a fine codice
            app.run(debug=True, host='127.0.0.1', port=5000)
            # in production, set debug to false
